# Budget Project

Set of applications for Budgets creation

### app

Android app

### backend

Backend Spring Boot server

- port: 8081

```
cd backend
mvn spring-boot:run
```

### db

Database information

### frontend

Vuejs frontend application

- port: 8080

```
cd frontend
npm run serve
```

### proxy

Node js proxy server interface

- port: 3000

```
cd mocks
node ./App.js
```

### postman

Postman collection
